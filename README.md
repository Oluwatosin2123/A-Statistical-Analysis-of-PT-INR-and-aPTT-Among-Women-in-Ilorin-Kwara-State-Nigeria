# A-Statistical-Analysis-of-PT-INR-and-aPTT-Among-Women-in-Ilorin-Kwara-State-Nigeria
Effects of Selected Oral Contraceptives on Coagulation Parameters
# Overview
This project presents a quantitative statistical analysis examining whether oral contraceptive (OC) use affects key blood coagulation parameters — Prothrombin Time (PT), International Normalized Ratio (INR), and Activated Partial Thromboplastin Time (aPTT) — among 80 women (35 OC users, 45 non-users).
The analysis combines descriptive statistics, One-Way ANOVA, and multivariate regression to test three research objectives and their corresponding hypotheses.
